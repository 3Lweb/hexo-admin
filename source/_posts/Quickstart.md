#### 1. Setup hexo & create a blog
```sh
npm install -g hexo
cd ~/
hexo init my-blog
cd my-blog
npm install
```
#### 2. Install the admin & start things up
```sh
npm install --save hexo-admin
hexo server -d
open http://localhost:4000/admin/
```
#### 3. Profit!
The UI should be pretty discoverable -- let me know if you can't find something.
